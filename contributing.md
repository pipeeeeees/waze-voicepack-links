# Contributing to Waze Voicepack Links

Thank you for your interest in contributing to the project! I appreciate your help in making the repository better.

## Ways to Contribute
1. Adding New Voice Pack URLs to the `readme.md` list:
   - Fork the repository.
   - Clone your fork to your local machine.
   - Add the new voice pack details to the `readme.md` fill in the appropriate section.
   - Commit your changes and push them to your fork.
   - Create a pull request to the main repository for review.

2. Requesting Help with Uploading mp3 files:
    - If you have a voice pack in mp3 format that you would like to see added to the list but are unsure how to upload it, please reach out to me by making an issue on the repository and attaching said mp3 files. I will assist you in getting them uploaded and added to the list when I have time
